---
title: Как установить моды на сервер
description: Установка модов на сервер позволяет всем игрокам использовать новые функции без необходимости настраивать каждый клиент по отдельности.
---

::: tip
Чтобы играть с модами на сервере, достаточно установить их только на сервере. На клиент ставить моды не нужно — они автоматически загрузятся с сервера.
:::

<ol class="vp-steps">
<li>
<strong>Остановка сервера</strong>
Перед добавлением новых файлов **обязательно выключите сервер**. Если изменять папку `Mods` при работающем сервере, изменения не вступят в силу или возникнет ошибка доступа к файлам.
</li>
<li>
<strong>Загрузка модификаций</strong>
Используйте только те моды, которые поддерживают работу на серверной стороне (Server-side).

- [CurseForge — Hytale Mods](https://www.curseforge.com/hytale)
</li>
<li>
<strong>Переход к папке сервера</strong>
Вам нужно попасть в директорию, где лежат исполняемые файлы сервера:

1. В консоли сервера нажмите кнопку **«Server Folder»**.
2. Или перейдите по пути: `%appdata%\HLauncher\ServerData`
</li>
<li>
<strong>Установка</strong>

1. Найдите внутри папку `mods`
   - _Примечание: Если папки нет, создайте её вручную с соблюдением регистра._
2. Скопируйте скачанные файлы модов в эту папку.
</li>
<li>
<strong>Запуск и проверка</strong>

1. Запустите сервер через лаунчер.
2. Внимательно следите за консолью: если какой-то мод не подходит по версии или конфликтует с другими, сервер выведет ошибку (Error) и остановится.
3. Если сервер запустился успешно (статус `Done`), моды активны для всех подключающихся игроков.
</li>
</ol>
