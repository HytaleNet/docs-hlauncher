---
title: Как включить PVP режим
description: По умолчанию на серверах Hytale урон между игроками может быть отключен. Чтобы активировать режим сражений, выполните следующие действия.
---

- Откройте **консоль сервера**.
- Введите следующую команду и нажмите Enter: `world settings pvp set true`

::: tip
Чтобы снова отключить урон между игроками, пропишите: `world settings pvp set false`
:::
